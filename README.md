# Code-Sandbox-Fiddle-Theme
Theme HTML, CSS & JS sandbox for developers to quickly try out code and share code snippets.

Awesome theme light & dark mode + responsive design.

Inspired by jsfiddle & codepen.

Built with:
- Codemirror
- Bootstrap
- Fontawesome
- jQuery
- jQuery.enhsplitter
- Google font-family
